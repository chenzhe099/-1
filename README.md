# 智慧农业管理系统

## 项目结构

```
gjcx/
├── index.html              # 主页面入口
├── README.md               # 项目说明文档
├── css/                    # 样式文件目录
│   └── styles.css          # 自定义CSS样式
├── js/                     # JavaScript文件目录
│   ├── app.js              # 主应用逻辑（侧边栏导航）
│   └── charts.js           # Chart.js图表初始化
├── assets/                 # 静态资源目录（图片、字体等）
├── data/                   # 数据文件目录（JSON数据等）
└── scripts/                # 辅助脚本目录
    ├── fix_*.py            # HTML修复脚本
    └── clean_*.py          # 数据清理脚本
```

## 功能模块

1. **数据总览与驾驶舱** - 农场全貌可视化展示，实时气象、土壤数据看板
2. **AI病虫害智能识别与预警** - 上传图片识别病虫害，显示防治建议
3. **AI精准农事决策（水肥药管理）** - 按地块生成个性化施肥、灌溉方案
4. **产量预测与农事规划** - 基于AI模型预测产量，智能排期
5. **数字化农场管理** - 全周期电子种植台账，多维度报表分析
6. **设备监控与远程控制** - IoT设备状态监控，远程控制操作
7. **农产品溯源管理** - 生产全过程追溯，生成溯源码
8. **权限管理与多账号协同** - 用户分级权限，操作日志管理

## 技术栈

- **HTML5** - 页面结构
- **Tailwind CSS** - 样式框架
- **Font Awesome** - 图标库
- **Chart.js** - 图表可视化
- **原生JavaScript** - 交互逻辑

## 运行项目

直接在浏览器中打开 `index.html` 文件，或使用本地服务器：

```bash
# 使用Python启动本地服务器
python -m http.server 8000
```

然后访问 `http://localhost:8000`
